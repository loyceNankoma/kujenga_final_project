# kujenga_final_project
This project implements a Susceptible-Exposed-Infectious-Recovered (SEIR) model to analyze and simulate infectious-disease dynamics using a Uganda-based dataset. The goal is to estimate transmission patterns, fit epidemiological parameters, and generate short-term projections to support data-driven public health insights.
The project integrates data preprocessing, exploratory analysis, model fitting, and simulation in Python.
# Objectives
Clean and structure epidemiological data from Uganda
Visualize trends in reported cases
Fit an SEIR model to observed data
Estimate key parameters (β, σ, γ)
Simulate disease spread dynamics
Generate short-term projections
